# Multiple Linear Regression from Scratch

A project implementing **Multiple Linear Regression** from scratch as part of **Lab 9 (Part 2)** of the **Data Science Theory and Practice (ED5340)** course at **IIT Madras**.

---

## 📌 Project Overview

This project explores and implements **Multiple Linear Regression** from the ground up — without relying on high-level ML libraries for the core model. The goal is to understand the mathematical foundations and apply them to a real-world dataset.

## 🔬 Key Concepts Covered

- Understanding **Multiple Linear Regression** mathematically
- Implementing the model from scratch using `numpy`
- Feature engineering and data preprocessing
- Computing **coefficients** using the **Normal Equation** or **Gradient Descent**
- Model evaluation using **R² Score**, **MSE**, **RMSE**
- Visualization of predictions vs actual values
